# 전국 척추 의료진 222명 통합분석 대시보드

단일 HTML 파일로 실행되는 전국 주요 척추 의료진 222명 통합분석 대시보드입니다.

## 실행

`index.html`을 최신 웹 브라우저에서 엽니다.

## 주요 기능

- 데이터 신뢰도, 진료과, 병원, 나이대, 졸업연대, 전문분야 차트
- 차트 클릭 시 해당 의료진 목록 표시
- 검색 및 다중 필터
- 전체 의료진 상세 테이블
- 검증 필요 의료진 분리
- 원본 기초 데이터 탭 및 CSV 다운로드

## 주의

대시보드에 포함된 나이·학력·소속·전문분야 일부는 추정 또는 보조자료 기반입니다. 실제 진료 선택 전 병원 공식 프로필을 다시 확인해야 합니다.
